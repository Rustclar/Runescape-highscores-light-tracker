# Release Notes - v2.1.0

## New

- RS3 Group Ironman action: group rank lookup with regular/competitive modes and team sizes 2–5.
- Property Inspector for GIM action with toggles for Rank/Total Level/XP.
- Text color and text size controls for both actions.
- Save + Test Pull buttons for the GIM action.

## Improvements

- GIM rank parsing aligned to list-page rank rows for accurate results.
- 10-minute TTL cache for group rank lookups to reduce repeated fetches.

## Notes

- End-user install: double-click the `.streamDeckPlugin` file from `dist/` or GitHub Releases.
